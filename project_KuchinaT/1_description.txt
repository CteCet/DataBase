БД представляет собой модель хранения данных веб-сайта https://linguatrip.com/ - 
платформа для бронирования языковых курсов в зарубежных школах.
БД состоит из 11 таблиц:
	- users, profiles c данными о пользователе
	- users_courses - языковые курсы, взятые пользователями с данными о
		датах поездки, с информацией о языковой школе, курсах, и вариантах проживания
	- online_edu - Online-курсы, предоставляемые платформой (Skype-уроки, Интесивы, Марафоны), 
	учителя, даты начала		
Данная БД решает следующие задачи:
	- предоставить пользователю выборку школ, основываясь на 
языке, стране, городе, курсе, цене, виде предоставляемого жилья
	- предоставить пользователю список Online-курсов доступных на платформе
	- создавать предложения с курсами для пользователей
	- подсчёт кол-ва пользователей и курсов
